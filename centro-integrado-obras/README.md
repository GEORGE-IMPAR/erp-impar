# ERP ÍMPAR — Centro Integrado de Gestão de Obras
## Pacote V10.1.7 — Agente Inteligente

Base utilizada: V10.1.6 estável.

### O que foi adicionado
- Aba **Agente** na Sala da Obra.
- Perguntas rápidas:
  - Coordenador
  - Percentual
  - Atraso
  - Participantes
  - Agenda de hoje
  - Mídias
  - Resumo da obra
- Respostas baseadas somente nos dados carregados do ERP e nos registros locais da sala.
- Quando não há dado seguro, o agente informa que não sabe.

### Regras preservadas
- ERP continua sendo a fonte oficial.
- Telegram segue como interface de campo.
- Uma obra = uma sala/RDO.
- Nenhum PHP novo neste pacote.
- Base V10.1.5/V10.1.6 estável preservada.
