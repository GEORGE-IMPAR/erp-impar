Como funciona
  1. Informe o código para editar ou crie um novo.
  2. Preencha as etapas. Ao clicar em Avançar, os dados são gravados.
  3. Você pode voltar depois: é só digitar o mesmo código.
  4. Ao concluir, use os botões para gerar os documentos.
